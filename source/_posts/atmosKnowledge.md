## 罗斯贝波速

罗斯贝波频率：
$$
\begin{align}
\omega = Uk - \beta \frac{k} { { {k^2} + {l^2} } }
\end{align}
$$
相速度和群速度：
$$
\begin{align}
c &= \frac{\omega } {k} = U - \frac{\beta } { { {k^2} + {l^2} } }\\\\
{c_g} &= \frac{ {\partial \omega } } { {\partial k} } = U - \frac{ {\beta ({l^2} - {k^2})} } { { { { ( {k^2} + {l^2})}^2} } }
\end{align}
$$
